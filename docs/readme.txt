Title                   : Satanic Gameshow Deluxe (v1.1)
Filename                : sgd_v11.pk3
Author                  : Santtu "MFG38" Pesonen
Email Address           : santtu.pesonen@mail.com

Description             : Satanic Gameshow Deluxe is a gameplay mod for
                          the classic Doom games, powered by GZDoom. The
                          mod's core feature set consists of a completely new
                          weapon arsenal, a "feat"-based weapon upgrade
                          system and finishers a la Wrack.
                                                    
                          Best played with your megawad of choice.

Additional Credits to   : See "credits.txt".
